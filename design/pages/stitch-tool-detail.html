New users of Claude Code are eligible for a one-time trial credit system. Upon initialization of the CLI tool, Anthropic provides $25 in free usage credits to test autonomous capabilities.

These credits are applied directly to your Console account and cover both input and output tokens. Because Claude Code is highly efficient with context management, this trial typically covers approximately 20-30 complex debugging sessions or feature implementations.

Frequently Asked Questions

Is it safe to give a CLI agent full shell access? expand_more
Claude Code operates with a "human-in-the-loop" default. For sensitive operations like deleting files or making network requests, it will prompt for explicit permission unless started in --yes mode.
Do I need my own Anthropic API key? expand_more
Yes, Claude Code currently requires an active Anthropic Console account. It uses your account balance to pay for tokens used during the session.
Can it work with legacy or undocumented code? expand_more
Claude Code is specifically designed for discovery. It can index massive legacy projects and use tool-calling to grep through files, building an internal mental model of how components interact.
